120_xy.exe 为校园招聘信息爬取程序
120_zx.exe 为最新招聘信息爬取程序

运行程序之前先确保电脑上已经安装  Chrome 浏览器并可以正常访问网络
（只适用于chrome版本为  61.0.3163.100（正式版本） （64 位））

	1.点击运行 ChromeWebDriver.exe
	2.点击运行 120_xy（zx）.exe 程序
	3.根据第一个提示输入 当天需要存档的信息的条数 
	例如 ：我要录取5号的信息，在学校就业网上点击按发布日期排序之后，5号共需要记录20条信息 那么在程序第一个提示输入20
	4.根据第二个提示输入开始页
	例如：排序之后发现，学校就业网上已经发到了7号的信息了，但是我只想录5号的信息，我在第3页发现第3页开始有5号的信息，那么在第二个提示处输入3
	5.根据第三个提示输入开始数
	例如：第三页从上往下数，第3条是6号的，第4条是5号的，那么在这里输入4
	6.运行程序 等待程序运行完成
	7.程序运行完成之后会在同级目录生成对应xls文件 ，就是我们生成的信息

注：
	1.程序运行会弹出一个可爱的工作中的浏览器框，千万不要点击它，让他自己工作，可以把它放一边做其他事情，一般工作5-10分钟就会结束。
	2.由于每个电脑网速不一，我只能加一些中断进去避免信息获取错位，所以运行过程会比较慢，请不要急躁。
	3.请正常的使用它。他没有任何异常测试，非常的脆弱，很容易就崩掉（如就业网出现异常网页），如果崩掉请重新启动程序，实在不行就手录。
	4.不接受大数据测试。（第一个输入建议大于0小于50，第二个输入建议大于0小于50，第三个输入0-20之间）
	5.使用之后记得检查生成的表格是否存在错位或者缺少字符。
	6.校园招聘表格无法生成校区信息，需要手动补。
	7.本程序仅支持爬取东北大学就业信息网内容，不可用于其他网站。
